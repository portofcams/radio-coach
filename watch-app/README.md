# Clearspar — Apple Watch micro-drills (watchOS app)

A wrist app with quick **phonetic / numbers / callsign** flashcard drills, fully
offline (no network, no Capacitor). The watchOS target is added to the iOS Xcode
project by a **script** — no manual GUI surgery needed — and it's verified to
compile against the watchOS SDK.

## Files
- `Sources/` — the app (committed, durable):
  - `WilcoWatchApp.swift` — `@main` app + `ContentView` (the drill menu)
  - `Drills.swift` — phonetic/number data + spelling logic (niner/fife/tree)
  - `DrillViews.swift` — the three flashcard drill screens
- `ios-template/` — `Info.plist` (WKApplication + companion bundle id) + a minimal
  `Assets.xcassets` (AppIcon/AccentColor placeholders).
- `../scripts/add-watch-target.rb` — stages the above into `ios/App/WilcoWatch/`
  and adds the `WilcoWatch` target (sources, build settings, Info.plist, embed-in-iOS
  phase, shared scheme).

## Wiring it up
`ios/` is gitignored and regenerated by Capacitor, so the wiring is a script you
re-run rather than commit:

```sh
gem install xcodeproj            # once (or: gem install --user-install xcodeproj)
npx cap sync ios                 # ensure ios/ exists
ruby scripts/add-watch-target.rb # adds + embeds the WilcoWatch target (idempotent)
```

Verify it compiles (no signing needed):

```sh
xcodebuild -project ios/App/App.xcodeproj -target WilcoWatch \
  -sdk watchsimulator -configuration Debug CODE_SIGNING_ALLOWED=NO build
# ** BUILD SUCCEEDED **   (verified 2026-06-20, Xcode 26.5 / watchOS 26.5 SDK)
```

## Before archiving to TestFlight
The target builds; these are the only archive-time touches left, done once in Xcode:
1. **App icon** — drop a 1024×1024 PNG into `WilcoWatch/Assets.xcassets/AppIcon.appiconset`
   (the slot exists; it's empty so the store build would otherwise be rejected).
2. **Signing** — `DEVELOPMENT_TEAM` is set to `CCSWC89S2V` with automatic signing;
   open the project once so Xcode creates the watch provisioning profile.
3. **Version** — `MARKETING_VERSION`/`CURRENT_PROJECT_VERSION` are set to match the
   app (1.0 / 6); bump both targets together for each new build.
4. Archive the **App** scheme as usual (the recipe in `ai-atc-radio-coach-plan.md`:
   Node22 PATH + UTF-8 + `AuthKey_SVA3JDJ975.p8`). The watch app embeds + rides along.

Pure SwiftUI + WatchKit haptic; no third-party deps. Bundle id
`com.binnacleai.radiocoach.watchkitapp` (companion of `com.binnacleai.radiocoach`).
